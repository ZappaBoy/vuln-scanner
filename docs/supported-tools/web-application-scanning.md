# Web Application Scanning

| Tool | Description |
|---|---|
| **Acunetix** | Commercial web vulnerability scanner that crawls and tests web apps for SQLi, XSS, and 7000+ other vulnerabilities. |
| **AppCheck Web Application Scanner** | Commercial platform for continuous web and network security scanning with detailed remediation guidance. |
| **AppSpider** | Rapid7's dynamic application security testing (DAST) tool that covers modern web apps including SPAs. |
| **Arachni** | Open-source, modular DAST framework designed for evaluating web application security. |
| **Burp Suite** | Industry-standard web security testing platform with proxy, scanner, and extensions; supports XML, API, GraphQL, DAST, and Dastardly output formats. |
| **Crashtest Security** | Automated DAST scanner focused on CI/CD integration for continuous web vulnerability detection. |
| **Drheader** | Lightweight tool that inspects HTTP response headers for security misconfigurations. |
| **HCL AppScan / ASOC SAST** | Enterprise DAST and SAST platform from HCL covering web, mobile, and open-source vulnerabilities. |
| **Immuniweb** | AI-powered web security testing platform combining DAST, SCA, and compliance checks. |
| **Invicti (Netsparker)** | Commercial DAST scanner with proof-based scanning to eliminate false positives in web apps. |
| **Microfocus WebInspect** | Enterprise-grade DAST tool from OpenText for deep scanning of complex web applications. |
| **Mozilla Observatory** | Free online tool that checks a website's HTTP headers, TLS, and security best practices. |
| **Nikto** | Open-source web server scanner that detects outdated software, misconfigurations, and known vulnerabilities. |
| **Outpost24** | Cloud-based vulnerability management platform covering web apps, APIs, and network assets. |
| **Rapplex** | Web application vulnerability scanner focused on ease of use and automated reporting. |
| **SkF** | Security Knowledge Framework — an OWASP project that provides security checklists and testing guidance aligned to ASVS. |
| **Solar AppScreener** | Russian-origin SAST/DAST hybrid tool supporting a wide range of programming languages. |
| **StackHawk** | DAST tool built for developers, designed to run in CI/CD pipelines and test APIs and web apps. |
| **Trustwave** | Commercial security testing platform offering web app scanning as part of its managed security services. |
| **Wapiti** | Open-source black-box web application scanner that audits for injections, XSS, file disclosure, and more. |
| **Wfuzz** | Open-source web fuzzer used to brute-force web application components such as parameters, directories, and forms. |
| **WPScan** | Open-source black-box WordPress vulnerability scanner that detects vulnerable plugins, themes, and configurations. |
| **ZAP (OWASP Zed Attack Proxy)** | The most widely used open-source DAST tool, maintained by OWASP, with both manual and automated scanning modes. |
