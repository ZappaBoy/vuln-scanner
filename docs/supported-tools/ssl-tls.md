# SSL/TLS

| Tool | Description |
|---|---|
| **SSLScan** | Open-source tool that quickly enumerates supported ciphers, protocols, and certificate details on TLS/SSL services. |
| **SSLyze** | Python-based TLS analyzer that checks for misconfigurations, weak ciphers, and known vulnerabilities like Heartbleed. |
| **SSL Labs** | Qualys's widely used online TLS grading service that assigns an A–F score based on protocol and configuration quality. |
| **testssl.sh** | Comprehensive open-source bash script for testing TLS/SSL configurations, ciphers, and protocol vulnerabilities from the command line. |
| **Humble** | Open-source HTTP headers analyzer focused on checking security headers and their compliance with best practices. |
