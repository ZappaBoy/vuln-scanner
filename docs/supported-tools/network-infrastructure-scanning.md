# Network & Infrastructure Scanning

| Tool | Description |
|---|---|
| **Nmap** | The de facto standard open-source tool for network discovery, port scanning, service detection, and OS fingerprinting. |
| **Nexpose** | Rapid7's on-premises vulnerability management solution for network and infrastructure assets. |
| **OpenVAS** | Open-source full-featured vulnerability scanner maintained by Greenbone, covering thousands of CVEs across network services. |
| **Qualys** | Cloud-based vulnerability management platform offering network scanning, web app testing, VMDR, and compliance modules. |
| **Tenable / Nessus** | Industry-leading vulnerability scanner covering networks, cloud, containers, and compliance across 70,000+ plugins. |
| **Cycognito** | External attack surface management platform that discovers and tests internet-exposed assets automatically. |
| **Hydra** | Fast open-source network login brute-forcer supporting dozens of protocols including SSH, FTP, HTTP, and SMB. |
| **SSH Audit** | Open-source tool that audits SSH server and client configurations for weak algorithms and known vulnerabilities. |
| **Wazuh** | Open-source SIEM and XDR platform with built-in vulnerability detection for hosts and infrastructure. |
| **Masscan** | Fastest TCP port scanner — sends SYN packets asynchronously at millions of packets per second. |
| **RustScan** | Modern port scanner written in Rust; finds open ports in seconds then pipes them to Nmap for service detection. |
| **Naabu** | Fast Go-based port scanner by ProjectDiscovery; designed to pair with Nuclei in recon pipelines. |
| **Zmap** | Stateless large-scale internet-wide network scanner; can scan the entire IPv4 address space in minutes. |
| **Amass** | In-depth attack surface mapping and asset discovery combining active/passive DNS, cert transparency, and APIs. |
| **Subfinder** | Passive subdomain discovery tool by ProjectDiscovery aggregating results from 40+ sources. |
| **Findomain** | Fast cross-platform subdomain enumerator using certificate transparency logs and multiple APIs. |
| **Massdns** | High-performance passive DNS bulk resolver for large-scale subdomain reconnaissance. |
| **Shuffledns** | Massdns wrapper that supports active bruteforce with wildcard and false-positive filtering. |
| **Puredns** | Fast, accurate DNS resolver with reliable wildcard detection and clean output. |
| **Bbot** | Recursive internet scanner with 100+ modules covering subdomains, ports, web, and OSINT. |
| **Assetfinder** | Lightweight tool to find related domains and subdomains from passive sources (Wayback, crt.sh, etc.). |
| **Dnsx** | Multi-purpose DNS toolkit by ProjectDiscovery for querying, bruteforcing, and chaining with other tools. |
| **Alterx** | Fast, customizable subdomain wordlist generator by ProjectDiscovery using permutation patterns. |
| **Dnsgen** | Powerful DNS name permutation and mutation tool for expanding subdomain wordlists. |
| **VHostScan** | Virtual host scanner with reverse lookup support to find hidden vhosts on shared IP addresses. |
| **Subdominator** | Fast subdomain enumeration aggregating 50+ passive sources with de-duplication. |
| **Chaos** | Go client for ProjectDiscovery's Chaos DNS API — bulk passive subdomain recon from a maintained dataset. |
| **Knockpy** | Python subdomain enumeration using wordlist-based dictionary attack and DNS queries. |
| **Sudomy** | Automated subdomain enumeration tool combining passive sources and active DNS resolution. |
| **HTTProbe** | Take a list of domains and probe for working HTTP and HTTPS servers; filters live hosts. |
