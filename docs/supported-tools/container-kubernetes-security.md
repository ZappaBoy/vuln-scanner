# Container & Kubernetes Security

| Tool | Description |
|---|---|
| **Anchore Engine / Enterprise / Grype / AnchoreCTL** | Open-source and commercial container image analysis suite that inspects images for CVEs, secrets, and policy violations. |
| **Aqua** | Commercial cloud-native security platform covering container images, running workloads, and Kubernetes clusters. |
| **Clair** | Open-source static vulnerability analyzer for container images, originally developed by CoreOS. |
| **Dockle** | Open-source container image linter that checks for security best practices and CIS benchmark compliance. |
| **DockerBench** | Open-source script that runs CIS Docker Benchmark checks against the host and running containers. |
| **Harbor Vulnerability** | Vulnerability scan results from Harbor, an open-source container registry with built-in security scanning. |
| **Kubeaudit** | Open-source CLI tool that audits Kubernetes clusters for common security misconfigurations. |
| **Kubebench** | Open-source tool by Aqua that runs CIS Kubernetes Benchmark checks on cluster nodes and control planes. |
| **Kubehunter** | Open-source Kubernetes penetration testing tool that hunts for security weaknesses in clusters. |
| **Kubescape** | Open-source Kubernetes security platform that checks clusters against NSA, MITRE, and CIS frameworks. |
| **Neuvector / NeuVector Compliance** | Open-source full lifecycle container security platform with runtime protection and compliance scanning. |
| **Sysdig (CLI, Reports)** | Commercial container and cloud security platform offering runtime threat detection and vulnerability management. |
| **Trivy / Trivy Operator** | Fast, open-source vulnerability and misconfiguration scanner for containers, filesystems, Git repos, and Kubernetes. |
| **Twistlock** | Palo Alto's (Prisma Cloud) container security platform for image scanning, runtime protection, and compliance. |
| **Zora** | Open-source Kubernetes multi-scanner that runs security checks and exposes results as Kubernetes custom resources. |
