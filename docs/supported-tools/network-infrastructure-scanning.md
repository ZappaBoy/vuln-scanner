# Network & Infrastructure Scanning

| Tool | Description |
|---|---|
| **Nmap** | The de facto standard open-source tool for network discovery, port scanning, service detection, and OS fingerprinting. |
| **Nexpose** | Rapid7's on-premises vulnerability management solution for network and infrastructure assets. |
| **OpenVAS** | Open-source full-featured vulnerability scanner maintained by Greenbone, covering thousands of CVEs across network services. |
| **Qualys** | Cloud-based vulnerability management platform offering network scanning, web app testing, VMDR, and compliance modules. |
| **Tenable / Nessus** | Industry-leading vulnerability scanner covering networks, cloud, containers, and compliance across 70,000+ plugins. |
| **Cycognito** | External attack surface management platform that discovers and tests internet-exposed assets automatically. |
| **Hydra** | Fast open-source network login brute-forcer supporting dozens of protocols including SSH, FTP, HTTP, and SMB. |
| **SSH Audit** | Open-source tool that audits SSH server and client configurations for weak algorithms and known vulnerabilities. |
| **Wazuh** | Open-source SIEM and XDR platform with built-in vulnerability detection for hosts and infrastructure. |
