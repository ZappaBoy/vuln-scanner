# Secret Scanning

| Tool | Description |
|---|---|
| **CredScan** | Microsoft tool that scans source code and build outputs for accidentally committed credentials and secrets. |
| **Detect Secrets** | Open-source Yelp tool that detects high-entropy strings and known secret patterns in codebases with a baseline workflow. |
| **GGShield** | GitGuardian's CLI tool that scans commits, branches, and CI pipelines for secrets against 350+ detectors. |
| **GitHub Secrets Detection** | GitHub's native secret scanning feature that alerts on detected credentials pushed to repositories. |
| **GitLab Secret Detection** | GitLab's built-in pipeline job that scans commits for secrets using regex-based and entropy-based rules. |
| **Gitleaks** | Open-source tool for detecting secrets in git history, files, and CI pipelines using customizable regex rules. |
| **Legitify** | Open-source tool by Legit Security that detects misconfigurations and security issues in GitHub and GitLab orgs. |
| **n0s1** | Open-source secret scanner designed to find leaked credentials in project management tools like Jira and Confluence. |
| **NoseyParker** | Open-source high-performance secret scanner that finds credentials in git history and large codebases at scale. |
| **Rusty Hog** | Open-source Rust-based secret scanner suite for S3, Git repos, Jira, and Confluence. |
| **Talisman** | Open-source git hook tool by Thoughtworks that prevents committing sensitive files and secrets to repositories. |
| **TruffleHog / TruffleHog3** | Open-source tool that scans git history and filesystems for high-entropy strings and known secret patterns. |
| **Whispers** | Open-source static analysis tool that identifies hardcoded secrets and dangerous behaviors in source code. |
