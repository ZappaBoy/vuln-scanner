# API / Platform Integrations

| Tool | Description |
|---|---|
| **BlackDuck API** | Synopsys Black Duck's REST API for pulling SCA findings and license compliance data into DefectDojo. |
| **Bugcrowd API** | Integration with Bugcrowd's bug bounty and vulnerability disclosure platform to import researcher-reported findings. |
| **Cobalt API** | Integration with Cobalt's pentest-as-a-service platform to pull findings from managed penetration tests. |
| **EdgeScan API** | Integration with EdgeScan's continuous vulnerability management platform for web and network asset findings. |
| **SonarQube API** | Pull-based integration with SonarQube's REST API to import code quality and security findings directly. |
| **Trustwave Fusion API** | Integration with Trustwave Fusion's managed security platform to import threat and vulnerability findings. |
| **Vulners API** | Integration with Vulners' vulnerability intelligence database API to enrich or import CVE-based findings. |
| **HackerOne (H1)** | Integration with HackerOne's bug bounty and VDP platform to import externally reported vulnerabilities. |
| **Intsights** | Integration with Intsights (now Rapid7) threat intelligence platform to import external threat findings. |
| **Risk Recon** | Integration with Risk Recon (now Mastercard) for third-party cyber risk ratings and external findings. |
