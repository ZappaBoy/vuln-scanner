# SAST (Static Analysis)

| Tool | Description |
|---|---|
| **Bandit** | Open-source Python SAST tool that finds common security issues by analyzing the AST of Python code. |
| **Bearer CLI** | Open-source SAST tool focused on detecting sensitive data flows and privacy risks in application code. |
| **Brakeman** | Open-source static analyzer for Ruby on Rails apps that detects a wide range of web vulnerabilities. |
| **Checkmarx (One, CxFlow, OSA)** | Enterprise SAST platform supporting 30+ languages with deep data-flow analysis and CI/CD integration. |
| **CodeChecker** | Open-source static analysis infrastructure built on top of Clang/LLVM tools for C and C++ codebases. |
| **Contrast** | Runtime application security platform (IAST) that instruments apps to detect vulnerabilities during testing. |
| **Coverity (API, Scan)** | Commercial SAST tool by Synopsys with deep inter-procedural analysis for C, C++, Java, and more. |
| **DawnScanner** | Open-source security code review scanner for Ruby, Rails, and Sinatra applications. |
| **ESLint** | Open-source JavaScript/TypeScript linter that can enforce security rules via plugins like `eslint-plugin-security`. |
| **Fortify** | OpenText's (formerly Micro Focus) enterprise SAST platform supporting 27+ languages with comprehensive vulnerability taxonomy. |
| **GitHub SAST** | GitHub's native code scanning feature powered by CodeQL for detecting vulnerabilities in repositories. |
| **GitLab SAST** | GitLab's built-in static analysis pipeline that runs multiple language-specific analyzers on merge requests. |
| **GoSec** | Open-source SAST tool that inspects Go source code for common security issues using AST analysis. |
| **HoruSec** | Open-source multi-language SAST CLI that orchestrates multiple analyzers and produces a unified report. |
| **HuskyCI** | Open-source security testing orchestrator for CI pipelines that runs multiple SAST tools and aggregates results. |
| **IBM App** | IBM's application security testing offering (formerly AppScan Source) for SAST across enterprise codebases. |
| **Kiuwan / Kiuwan SCA** | Commercial code analysis platform covering SAST and SCA with support for 30+ programming languages. |
| **PMD** | Open-source multi-language static analyzer that detects code quality issues and some security anti-patterns. |
| **ProgPilot** | Open-source PHP SAST tool that performs taint analysis to detect injection vulnerabilities. |
| **PWN SAST** | Lightweight SAST scanner focused on detecting hardcoded credentials and common vulnerability patterns. |
| **RuboCop** | Open-source Ruby linter and formatter that includes security-focused cops via `rubocop-ast`. |
| **Semgrep / Semgrep Pro** | Open-source and commercial lightweight SAST tool using pattern-matching rules across 30+ languages. |
| **SonarQube / SonarQube API** | Open-source and commercial code quality and security platform with SAST capabilities across 30+ languages. |
| **SpotBugs** | Open-source Java bytecode analyzer (successor to FindBugs) that detects bug patterns including security issues. |
| **Veracode / Veracode SCA** | Commercial cloud-based SAST and SCA platform with binary analysis support and detailed remediation guidance. |
| **Xanitizer** | Commercial Java and Scala SAST tool focused on web security vulnerability detection via data-flow analysis. |
