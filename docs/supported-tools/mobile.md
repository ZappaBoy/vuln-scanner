# Mobile

## Frameworks & All-in-One

| Tool | Description |
|---|---|
| **MobSF / MobSF Scorecard** | Open-source Mobile Security Framework for automated static and dynamic analysis of Android and iOS applications. |
| **Drozer** | Security testing framework for Android — CLI agent performs extensive package, activity, and attack surface enumeration. |
| **Mobile Audit** | Web application for performing static analysis and detecting malware in Android APKs. |
| **MARA** | Mobile Application Reverse Engineering and Analysis Framework combining multiple tools. |

## Static Analysis

| Tool | Description |
|---|---|
| **AndroBugs** | Efficient Android vulnerability scanner that checks APKs for security issues and anti-patterns. |
| **QARK** | Quick Android Review Kit — static analysis tool for Android application security vulnerabilities. |
| **APKiD** | Identify packers, protectors, obfuscators, and other oddities in Android APK files. |
| **APKLeaks** | Scan APK files for embedded URIs, endpoints, and secrets using regex-based analysis. |
| **Androwarn** | Static code analyzer for Android — detects potential malicious behaviors like data leakage and telephony abuse. |
| **Quark-Engine** | Android malware scoring system using obfuscation-tolerant behavior analysis. |
| **StaCoAn** | Cross-platform static code analysis tool for Android (and iOS) applications. |
| **JAADAS** | Joint Advanced Defect Assessment for Android — flow-sensitive static analysis for vulnerabilities. |
| **Detekt** | Static code analysis for Kotlin — catches security anti-patterns in Android source code. |

## Reverse Engineering & Decompilation

| Tool | Description |
|---|---|
| **Jadx** | Dex-to-Java decompiler with a GUI and CLI; produces readable Java source from APK/DEX files. |
| **Androguard** | Reverse engineering and analysis framework for Android applications — Python library and CLI. |
| **Dex2Jar** | Tools to work with Android `.dex` and Java `.class` files — convert DEX to JAR for decompilation. |
| **Bytecode Viewer** | Lightweight multi-decompiler Java bytecode viewer supporting Android DEX files. |
| **Smali/Baksmali** | Assembler/disassembler for the DEX format used by the Android Dalvik VM. |
| **Apktool** | Reverse engineering tool for Android APKs — decode resources and reassemble modified APKs. |
| **Enjarify** | Translate Dalvik bytecode to equivalent Java bytecode for use with Java analysis tools. |
| **Krakatau** | Java decompiler, assembler, and disassembler with robust handling of obfuscated bytecode. |
| **Radare2** | Unix-like reverse engineering framework and command-line toolset; supports ARM/Dalvik. |
| **One Step Decompiler** | Python3 script that fully automates the process of decompiling Android APK files. |
| **Redexer** | Reengineering tool for Android app binaries — supports transformation and analysis. |
| **Simplify** | Generic Android deobfuscator that simplifies obfuscated smali code. |
