# Generic / Multi-format

| Tool | Description |
|---|---|
| **SARIF** | Static Analysis Results Interchange Format — a universal JSON standard for exchanging static analysis tool output. |
| **Generic Findings** | DefectDojo's built-in generic parser for importing simple CSV or JSON finding reports from custom or unsupported tools. |
| **OpenReports** | Open-source reporting framework that aggregates output from multiple security tools into unified reports. |
| **OpenSCAP** | Open-source implementation of the SCAP standard for auditing system compliance against security policies and benchmarks. |
| **Garak** | Open-source LLM vulnerability scanner that probes large language models for prompt injection, hallucination, and other risks. |
| **Threat Composer** | AWS open-source threat modeling tool that generates structured threat statements exportable as findings. |
| **JFrog Xray** | Commercial security and compliance tool integrated into the JFrog platform for scanning artifacts, containers, and builds. |
| **ReversingLabs SpectraAssure** | Commercial binary analysis platform that inspects software packages for malware, vulnerabilities, and supply chain risks. |
| **Xygeni** | Commercial software supply chain security platform covering code, dependencies, secrets, IaC, and CI/CD pipeline integrity. |
