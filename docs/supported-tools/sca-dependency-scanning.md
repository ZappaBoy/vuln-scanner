# SCA / Dependency Scanning

| Tool | Description |
|---|---|
| **Bundler Audit** | Open-source gem that checks Ruby Gemfile.lock for known vulnerabilities using the ruby-advisory-db. |
| **cargo-audit** | Open-source Rust tool that audits Cargo.lock for crates with known security vulnerabilities from RustSec. |
| **CycloneDX** | OWASP standard SBOM format; tools generating CycloneDX BOMs can be imported for vulnerability correlation. |
| **Dependency Check** | OWASP open-source tool that identifies known CVEs in project dependencies across Java, .NET, Python, and more. |
| **Dependency Track** | OWASP open-source continuous SBOM analysis platform that monitors component vulnerabilities over time. |
| **GitLab Dependency Scan** | GitLab's built-in SCA pipeline that scans project dependencies for known vulnerabilities on every commit. |
| **Govulncheck** | Official Go tool from the Go team that reports known vulnerabilities in Go module dependencies. |
| **Mend (WhiteSource)** | Commercial SCA platform with a large vulnerability database, license compliance, and auto-remediation PRs. |
| **Meterian** | Commercial multi-language SCA tool that checks dependencies for CVEs and license issues in CI/CD. |
| **Nancy** | Open-source Sonatype tool for auditing Go modules against the OSS Index vulnerability database. |
| **npm Audit / npm Audit v7+** | Built-in npm command that checks Node.js dependencies against the npm security advisory database. |
| **NSP** | Legacy Node Security Project CLI (predecessor to npm audit) for checking Node.js package vulnerabilities. |
| **ORT** | OSS Review Toolkit — open-source tool for analyzing project dependencies, licenses, and known vulnerabilities. |
| **OSS Index (DevAudit)** | Sonatype's free vulnerability database and DevAudit CLI tool supporting multiple package managers. |
| **OSV Scanner** | Open-source Google tool that queries the OSV database for vulnerabilities across multiple language ecosystems. |
| **pip-audit** | Open-source Python tool that audits installed packages and requirements files against PyPI advisory data. |
| **RetireJS** | Open-source tool that detects JavaScript libraries with known vulnerabilities in web apps and Node projects. |
| **Snyk / Snyk Code / Snyk Issue API** | Commercial developer-first security platform combining SCA, SAST, container, and IaC scanning. |
| **Sonatype** | Commercial SCA platform (Nexus IQ) with deep component intelligence and policy enforcement for pipelines. |
| **Xeol** | Open-source tool that detects end-of-life (EOL) software components in SBOMs and container images. |
| **Yarn Audit** | Built-in Yarn command that checks JavaScript dependencies for known vulnerabilities using the npm advisory DB. |
