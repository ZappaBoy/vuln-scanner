# IaC / Cloud Security

| Tool | Description |
|---|---|
| **AWS Inspector v2** | AWS-native vulnerability management service that scans EC2 instances and container images for CVEs and network exposure. |
| **AWS Prowler / Prowler v3+** | Open-source tool that audits AWS, Azure, and GCP environments against CIS benchmarks and security best practices. |
| **AWS Security Hub (ASFF)** | AWS aggregation service for security findings from multiple AWS services and third-party tools, using the ASFF format. |
| **Azure Security Center** | Microsoft Defender for Cloud — provides security posture management and threat protection across Azure resources. |
| **Checkov** | Open-source IaC scanner by Bridgecrew that checks Terraform, CloudFormation, Kubernetes, and ARM templates for misconfigurations. |
| **Chef Inspect** | InSpec-based compliance scanning that tests infrastructure state against policy-as-code profiles. |
| **Cloudsploit** | Open-source cloud security scanner that detects misconfigurations across AWS, Azure, GCP, and Oracle Cloud. |
| **Cloudflare Insights** | Security findings and analytics surfaced through Cloudflare's platform for assets behind its network. |
| **DSOP** | DoD DevSecOps Platform hardening checks for container images used in US government environments. |
| **KICS** | Open-source Checkmarx tool that detects security vulnerabilities and misconfigurations in IaC files across 24+ platforms. |
| **KrakenD Audit** | Security audit tool for KrakenD API gateway configurations, detecting unsafe settings and exposure risks. |
| **Orca Security** | Agentless cloud security platform that uses SideScanning to detect vulnerabilities, misconfigs, and risks across cloud assets. |
| **Popeye** | Open-source Kubernetes cluster sanitizer that reports on misconfigured or potentially unsafe resource definitions. |
| **Scout Suite** | Open-source multi-cloud auditing tool that assesses the security posture of AWS, Azure, GCP, and more. |
| **Terrascan** | Open-source IaC static analysis tool by Tenable that checks Terraform, Helm, and Kubernetes for policy violations. |
| **Terraform (tfsec)** | Open-source static analysis tool for Terraform code that detects misconfigurations and security issues before deployment. |
| **Threagile** | Open-source agile threat modeling tool that generates risk findings and attack trees from a YAML model definition. |
| **Wiz / WizCLI (Dir, IAC, Image)** | Commercial cloud security platform with agentless scanning for misconfigurations, CVEs, and secrets across cloud environments. |
